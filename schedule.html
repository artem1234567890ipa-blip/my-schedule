⬆ Прогрессивная нагрузка

  • Каждую неделю +1–2 повтора к каждому подходу
  • Достиг 20 повт. → усложни движение
  • Медленный темп (3–4 сек вниз) = нагрузка без веса
  • Веди дневник — без записей нет прогресса
  • 3 недели без роста = пора менять упражнение

🍽 Питание для роста

  • Белок: 1.6–2.2 г/кг = 112–154 г в день
  • Яйца, творог, курица, рыба, бобовые
  • Профицит +200–300 ккал/день для массы
  • Вода: 2–2.5 литра в день минимум
  • Сон 8–9 часов — основной фактор восстановления
